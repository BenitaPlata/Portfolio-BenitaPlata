Mis proyectos

Estos son algunos de los proyectos que he desarrollado para practicar, mejorar mis habilidades y construir mi camino como desarrolladora web.

Mi portfolio

Portfolio creado con HTML y CSS, diseñado con un enfoque UX que prioriza la claridad y la armonía visual. Su estructura responsive se adapta a cualquier dispositivo, ofreciendo una navegación fluida y cómoda.

BubblEase

Aplicación interactiva creada en React que genera burbujas animadas mediante un sistema de partículas personalizado. Incluye animaciones CSS optimizadas y diseño totalmente responsive.

MontessoriPlay

Frontend desarrollado con React y Context API, con CRUD funcional, búsqueda, preview de imágenes y layout responsive. Proyecto diseñado con estética Montessori y desplegado en Vercel.

Portal de sesiones (Backend)

Aplicación backend desarrollada con Node.js y Express, con renderizado en servidor mediante EJS. Incluye registro y login de usuarios, gestión de sesiones, carrito en sesión, preferencias visuales con cookies y persistencia de datos en ficheros JSON.

Vanilla JS Session Portal

Aplicación frontend desarrollada con JavaScript puro que carga sesiones dinámicamente, gestiona un carrito persistente con localStorage y permite personalizar la experiencia visual del usuario, sin backend.

Quickplan

Aplicación web desarrollada con React y Vite para la gestión de eventos. Permite consultar eventos desde un archivo JSON, aplicar búsqueda por texto, filtrar por categoría y gestionar una lista de favoritos, con una interfaz clara y funcional.
